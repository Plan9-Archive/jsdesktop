Javascript Desktop: A Backend-less Proof-of-Concept
Press New button to get a new javascript shell window. When you type javascript into the input textarea and press enter, three things happen:
  • The text you typed goes to the output textarea.
  • The text also gets run with eval(). No guardrails.
  • If eval was successful, the input is cleared to accept more javascript.
Any javascript your browser supports in eval should work. Be careful and enjoy.
